应用程序安装

  1. 通过RPM安装VSFTPD，同时查看软件信息并截图
  2. 通过YUM安装GCC软件，并查看GCC版本号，截图为准
  3. 通过DNF安装HTTPD软件，将成果截图

1.安装VSFTPD，查看软件信息

mount /dev/cdrom /mnt
cd /mnt/AppStream/Packages
rpm -ivh vsftpd-3.0.3-28.el8.x86_64.rpm

查看软件信息

rpm -qi vsftpd

2.安装GCC，查看软件信息

cd /etc/yum.repos.d
touch server.repo
vim server.repo

yum -y install gcc

查看GCC版本号

gcc --version

3.通过DNF安装HTTPD软件

yum install dnf 
dnf install httpd
